# 기본 형식

1. 일체식
    1. 설명
        1. C 언어 이전의 프로그래밍 스타일
        1. 프로그래머가 깊이 생각할 시간이 없음을 증명하는 코드
    1. 제약
        1. 명명된 추상화가 없다
        1. 라이브러리를 거의 또는 전혀 사용하지 않는다


2. 요리책
    1. 설명
        1. 사이드 이펙트가 존재한다
        1. 프로시저는 멱등이 아니다 
    1. 제약
        1. 먼 분기가 없다
        1. 절차적 추상화가 가능하다
        1. 프로시저는 상태를 전역 변수 형태로 공유 가능하다
        1. 규모가 큰 문제는 공유한 상태를 변경하거나, 상태를 더 추가하는 프로시저를
            이용하여 해결한다
